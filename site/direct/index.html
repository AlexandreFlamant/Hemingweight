Install Hemingweight to continue

You're seeing the pager because Hemingweight isn't running on your machine yet. The app is local. Open Terminal and paste this:

curl -fsSL https://raw.githubusercontent.com/AlexandreFlamant/hemingweight/main/install-remote.sh | bash

Takes about two minutes. One password prompt to trust the local HTTPS cert. When it finishes this page switches to the real app automatically. If the status dot at the top-left stays red after install, see the troubleshooting notes.
