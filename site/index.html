Get started

Install Clawable in 5 minutes

1

Open Terminal

Press Cmd + Space to open Spotlight, type Terminal, and press Enter.

2

Run the installer

Copy and paste this command into Terminal, then press Enter.

curl -fsSL https://raw.githubusercontent.com/AlexandreFlamant/clawable/main/install-remote.sh | bash
3

Install the Chrome extension

Download the extension, unzip it, then load it into Chrome.

Download Chrome Extension

Then open Chrome and go to chrome://extensions, enable Developer Mode (top right toggle), click Load Unpacked, and select the unzipped folder.

4

Sign in to Claude

Click the Clawable extension icon in your Chrome toolbar. On first launch, you'll be prompted to sign in to your Claude account.

5

Start building

Type what you want to build and hit Enter. That's it.
